3Ô∏è‚É£ Inside the `.git` Folder

Git creates a `.git` folder that contains all repository data, including:

  • HEAD: Points to the current branch.
  • objects/: Stores commits, blobs, trees.
  • refs/: Contains branch references.
  • config: Repository settings.
